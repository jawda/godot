# godot2d — learning game dev

A collection of Godot 4 projects built while learning 2D game development.

---

## Projects

### 1. Platform Jumper *(completed)*
A Mario-style side-scrolling platformer. Built first to get a feel for Godot's node system, physics, and input handling.

### 2. ARPG *(completed)*
A top-down Zelda-style RPG. Explored 2D world design, camera follow, basic combat, and tilemaps.

### 3. Roguelike TCG — `roguelike-tcg/` *(active)*
A roguelike deck-builder in the style of Slay the Spire. Dark fantasy aesthetic — gothic, pixel art inspired, morally complex.

Full game flow is implemented: start screen → character select → battlefield combat. The Cleric is the first playable class with a 21-card draft pool, stats system, and full combat including status effects, enemy phases, and a summon mechanic.

See [`roguelike-tcg/README.md`](roguelike-tcg/README.md) for full details on the project structure, systems, and how to extend the game.
